# Multi-Capsule Pipeline Analysis

## 📌 Overview
This project focuses on the **hydrodynamic performance** and **reliability analysis** of multi-capsule pipeline transport systems. The study uses statistical methods to evaluate how different operational parameters affect system efficiency and failure probability.

The analysis is conducted using Minitab, applying regression models, ANOVA, and reliability techniques.

---

## 🎯 Objectives
- Analyze the effect of flow parameters on pressure drop
- Evaluate capsule transport efficiency
- Identify significant factors influencing system performance
- Assess system reliability and failure probability
- Determine optimal operating conditions

---

## ⚙️ Parameters Studied

### Input Variables:
- Flow Velocity (m/s)
- Capsule Size (m)
- Capsule Spacing (m)
- Fluid Properties (if applicable)

### Output Variables:
- Pressure Drop (kPa)
- Energy Loss
- Capsule Velocity
- Failure Occurrence (0 = No Failure, 1 = Failure)

---

## 📊 Methods Used
- Descriptive Statistics
- Regression Analysis
- ANOVA (Analysis of Variance)
- Binary Logistic Regression
- Reliability Analysis (Failure Modeling)

## 📈 Key Findings (Example)
- Flow velocity significantly increases pressure drop
- Larger capsule sizes increase system resistance
- Increased spacing improves flow efficiency
- Failure probability increases at higher velocities
